# JavaScript Variables Practice (var, let, const)

## 📘 Overview
This repository contains 40 JavaScript examples demonstrating the behavior of:
- `var`
- `let`
- `const`

Each snippet includes:
- ✅ Code
- 📤 Output
- 🧠 Reason

This project is useful for:
- Beginners learning JavaScript
- Interview preparation
- Understanding variable scope and errors

---

## 🎯 Concepts Covered

### 🔹 var
- Allows redeclaration
- Allows reassignment
- Function scoped

### 🔹 let
- Does NOT allow redeclaration
- Allows reassignment
- Block scoped

### 🔹 const
- Does NOT allow redeclaration
- Does NOT allow reassignment
- Must be initialized at declaration

---

## ⚠️ Important Notes
- Some examples intentionally produce errors ❌
- These errors help understand JavaScript rules better
- Execution may stop at the first error if run fully
